# Anime Watchlist (Bright UI)

A bright, modern single-page anime/media watchlist app.

## Features

- Add anime, movies, TV/web series, manga, and more.
- Store season/episode counts, year, genre, studio, status, rating, image URL, and watch URL.
- Search and filter by media type.
- Data persistence with LocalStorage.

## Run locally

Open `index.html` directly in your browser.

## Publish (GitHub Pages)

This repository includes a workflow at `.github/workflows/pages.yml`.

1. Push the repo to GitHub on the `main` branch.
2. In **Settings → Pages**, ensure source is **GitHub Actions**.
3. Push any commit to `main` (or run the action manually).
4. Your site will be published at your GitHub Pages URL.
